How it works

The model looks at different characteristics of the URL and predicts whether it is likely to be a phishing website or a legitimate one.

Step 1
Input URL
Paste the website link you want to check into the detector.
Step 2
Feature Extraction
We extract features such as length, special characters, and domain patterns.
Step 3
Model Prediction
The trained model predicts if the URL is phishing or safe and shows the result.

About the Project

This phishing website detection system is built as a mini project to demonstrate how machine learning can be used for cybersecurity.

The backend is developed using Python (Flask) and a trained ML model(Random Forest and XGboost). The model is trained on a dataset of phishing and legitimate URLs, using features like URL structure and domain information.
